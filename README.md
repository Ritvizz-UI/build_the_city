# 🏙️ Build a City — SDLC Game

An interactive web-based game that helps users understand the **Software Development Life Cycle (SDLC)** in a fun and visual way.

---

## 🎯 Project Description

This project is an educational quiz-based game where users answer questions from different SDLC phases.
Each correct answer contributes to building a virtual city, showing progress from an empty land to a fully developed city.

---

## 🧠 SDLC Phases Covered

* Planning
* Requirements Analysis
* System Design
* Implementation
* Testing & Maintenance

---

## 🎮 Features

* Interactive quiz system
* Score tracking
* Phase-wise progression
* Visual city-building simulation
* Immediate feedback on answers

---

## 🛠️ Technologies Used

* HTML (Structure)
* CSS (Styling)
* JavaScript (Game Logic)
